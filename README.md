# Luvit for Mavericks (OS X 10.9)

b.sh is the build configuration script.

libuv_mavericks.patch is self descriptive.

buvit.sh is for creating luvit and friends but not luvi quickly.

# Notes

This runs libuv 1.49.0, and at the time of writing the master luvi, lit and stable luvit.

lit has the patches from Bilal2453 for luvit-loader refactor.

libuv has homemade patch for Mavericks.
